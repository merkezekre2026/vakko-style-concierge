# Vakko Style Concierge — Android Prototype

Premium Vakko müşterileri için kişisel stil asistanı konseptinin Jetpack Compose prototipi.

## İçerik

- Etkinliğe göre dinamik kombin seçkisi
- Vakko Moments özel gün akışı
- Sanal gardırop ve yeni ürün eşleştirmesi
- Stil danışmanı randevu girişi
- AR deneme ve premium ayrıcalık girişleri
- Dört sekmeli, erişilebilir mobil navigasyon

## Çalıştırma

Projeyi Android Studio Ladybug veya daha yeni sürümde açın. JDK 17 ve Android SDK 35 ile Gradle senkronizasyonunu tamamlayıp `app` yapılandırmasını çalıştırın.

> Bu sürüm, ürün kataloğu ve müşteri verilerini örnek veriyle gösteren UI prototipidir. Üretim sürümünde Vakko katalog/CRM API'leri, randevu servisi ve seçilen AR SDK'sı bağlanmalıdır.
